# Quick Reference Guide

## What Model Should I Use?

Select your use case to find the right model for your needs.

### Planning Applications
- **Burn Probability**: FSim, FlamMap
- **Fuel Treatment**: IFTDSS, FlamMap
- **Risk Assessment**: FARSITE, FlamMap

### Real-Time Operations
- **Active Fire Spread**: WRF-Fire, ELMFIRE
- **Evacuation Planning**: FARSITE, ELMFIRE
- **Resource Allocation**: FSPro

### Research Applications
- **Physics-Based**: FIRETEC, WFDS
- **Coupled Models**: WRF-Fire, CAWFE
- **Machine Learning**: Custom implementations
